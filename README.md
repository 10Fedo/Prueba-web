# Prueba Web

Página web creada como tarea para alojar una web en GitHub Pages.

## Tecnologías usadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Cambio de imágenes al hacer clic
- Campo de texto interactivo
- Objeto en movimiento con JavaScript

## Estética
Inspirada en la web de El País sobre el cambio climático.
